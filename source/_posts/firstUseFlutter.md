最近在寻找新的 GUI 框架，正好看到了 **Flutter，于是就尝试了一下。**

**Flutter** 可以支持极快的热重载技术，并且还支持超多平台。

同时 **Flutter** 使用 **Dart** 语言，这是一个非常方便的语言，它的语法简单，功能强大。
并且 **Flutter** 的编写也很舒服，它的 UI 组件库也很完善。

配置 **Flutter** 的环境可在 [Flutter 官方文档](https://docs.flutter.dev/install) 中找到。

本人也试着用 **Flutter** 写了一个简单的项目，名字叫做 [Pan123Next](https://github.com/123panNextGen/Pan123Next)。

**Flutter** 有着 `Material 3` 和 `Cupertino 3` 两种 UI 设计语言，你可以根据自己的需求来选择。
并且 **Flutter** 也有着 `FluentUI` 的 package。

**Flutter** 利用 `pub.dev` 来管理依赖，具有丰富且完善的 package 库。

**Flutter** 也有许多优秀的项目，例如 [FlClash](https://github.com/chen08209/FlClash) 和 [Pilipili](https://github.com/bggRGjQaUbCoE/PiliPlus)
